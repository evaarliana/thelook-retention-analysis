# TheLook Retention Analysis

## Overview

This project analyzes inventory growth and customer retention behavior using Google BigQuery SQL.

The objective is to understand inventory movement across product categories and evaluate customer retention patterns through cohort analysis.

## Business Questions

* How does inventory grow over time across product categories?
* Which categories experience significant inventory fluctuations?
* What is the monthly customer retention rate?
* How many customers return after their first purchase?
* What insights can be derived from customer cohort behavior?

## Tools

* Google BigQuery SQL

## Analysis Methods

* Common Table Expressions (CTE)
* Window Functions (LAG)
* Cohort Analysis
* Retention Analysis
* Growth Rate Analysis
* Date Functions

## Key Findings

* Inventory growth fluctuates across product categories rather than showing consistent growth.
* Several categories experience significant inventory changes over time.
* Customer retention drops sharply after the first purchase.
* Most cohorts retain only a small percentage of customers in subsequent months.
* Long-term customer engagement remains a key business challenge.

## Project Deliverables

* Inventory Growth Analysis
* Cohort Retention Analysis
* Business Insights
* Strategic Recommendations
